# Flask Chatbot

This is a Flask-based chatbot application that uses AIML (Artificial Intelligence Markup Language) for natural language processing and Neo4j as a graph database for storing user information.

## Prerequisites

- Python 3.x
- Flask
- py2neo
- pandas
- numpy
- matplotlib
- nltk
- requests
- BeautifulSoup
- transformers
- tensorflow
